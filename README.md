
# Edufree

An android app for sharing free courses over terabox and getting latest news about tech through blogs 

This app also have an global chat and comments for users to communicate about related topic 


## Tech Used

- Java
- FireBase Cloud
- Firebase Storage
- Android Material UI
- Picasso
- Chip Navigation Bar
